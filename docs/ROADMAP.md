# План развития (Roadmap)

## Версия 0.2 — Ближайшие улучшения (Текущий фокус)
- [ ] **Хаб:** Вывести MAC-адрес хаба и узла прямо на веб-страницу для удобства настройки.
- [ ] **Хаб:** Добавить минимальные улучшения CSS (кнопки, отступы, шрифты) для лучшего вида.
- [ ] **Хаб:** Зафиксировать версию библиотеки `AsyncTCP` в `platformio.ini`, добавив `ESP32 Async TCP @ 1.2.2`.
- [ ] **Узел:** Добавить датчик температуры/влажности DHT22 и отправку данных раз в 60 секунд на хаб.
- [ ] **Общее:** Прописать явно версию платформы `platform = espressif32 @ 6.12.0` в `platformio.ini` для полной воспроизводимости.

## Версия 0.3 — Расширение функциональности
- [ ] **Хаб:** Добавить поддержку управления 2-3 узлами (выбор из выпадающего списка на веб-странице).
- [ ] **Хаб:** Добавить веб-форму для отправки произвольных текстовых команд.
- [ ] **Система:** Разработать простой протокол для запроса данных с узлов (например, команда `GET_TEMP`).
- [ ] **Узел:** Реализовать хранение и отправку состояния нескольких пинов/датчиков.

## Версия 0.4 — Надежность и мониторинг
- [ ] **Хаб:** Добавить в веб-интерфейс индикатор статуса связи (онлайн/офлайн) для каждого узла.
- [ ] **Хаб:** Реализовать простое логирование основных событий (подключение, команды, ошибки) в Serial.
- [ ] **Узел:** Добавить механизм повторной отправки подтверждения (`ACK`), если первая попытка не удалась.
- [ ] **Документация:** Составить схему подключения датчиков и принципиальную схему в папке `docs/hardware/`.
